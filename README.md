# React + TypeScript + Vite

## Objetivo

O objetivo deste projeto é avaliar o código e design quanto à funcionalidade, usabilidade e aderência aos requisitos estabelecidos.

## API

A API utilizada é a Google Book API.

## Instalação

```bash
$ pnpm install
$ pnpm run dev
```
